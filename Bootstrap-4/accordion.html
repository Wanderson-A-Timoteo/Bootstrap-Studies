Accordion

Diferentemente do collapse este modelo, mostra o conteudo ou esconde, porém quando clicado para mostrar o conteúdo de um botão, quando clicamos em outro botão para ver o seu conteúdo, automaticamente o conteúdo anterior e escondido, ou seja, só mostra o conteúdo de apenas um botão por vez.

Neste exemplo vamos criar um botão tipo link que ao ser clicado mostrará o conteúdo ou esconderá o conteúdo. Para fazer este efeito precisamos definir o data-toggle="collapse" e indicar qual será o conteúdo usando o ID que esta classe data-toggle="collapse" vai alterar para esconder ou para mostrar usando data-target="ID do conteúdo referenciado".

Ainda precisamos definir que o boão será o responsável de esconder o conteúdo em ID com aria-controls=c1, neste não podemos colocar #.
