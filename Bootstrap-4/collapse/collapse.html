Collapse

Neste exemplo vamos criar um botão que ao ser clicado mostrará ou esconderá uma parte de conteúdo definida. Para fazer este efeito precisamos definir o data-toggle="collapse" e indicar qual será o conteúdo usando o ID que esta classe data-toggle="collapse" vai alterar para esconder ou para mostrar usando data-target="ID do conteúdo referenciado"

Este é o texto que vai aparecer ao clicar no botão. Mas po ser uma foto, até mesmo uma página inteira.
